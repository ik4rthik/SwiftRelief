## 📖 About

SwiftRelief is an AI-powered hospital recommendation system developed as a **B.Tech Mini Project**.
The application helps users discover nearby hospitals intelligently based on location, urgency, and healthcare requirements.

The project combines:

* 📱 Cross-platform mobile development using Flutter
* 🧠 Machine Learning integration using Python
* 📍 Location-aware hospital recommendations
* ⚡ Real-time backend communication through APIs

---

## ✨ Features

* Smart hospital recommendation system
* User authentication & profile management
* AI-assisted symptom mapping
* Location-aware search
* Responsive Flutter frontend
* REST API backend using Flask
* Machine Learning-based ranking logic
