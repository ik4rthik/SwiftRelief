# SwiftRelief Flutter 📱

Frontend application for the SwiftRelief intelligent hospital recommendation system.

## ✨ Features
- User-friendly mobile interface
- Real-time hospital recommendations
- Location-aware services
- Emergency-focused design
- Offline support capabilities

## 🛠️ Technologies Used
- Flutter
- Dart
- Material Design

## 🚀 Getting Started

```bash
flutter pub get
flutter run
```
